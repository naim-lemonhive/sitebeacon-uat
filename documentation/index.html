Installing Development Environment

We require many packages (dependencies) to run our site. Move into /azzle-next (project root) directory and run command below. 

npm install

After the installation, you should have a folder named /node_modules in the project root directory..

How To Run Development Server

To start our development server run command below:

npm run dev

Open your browser and visit http://localhost:3000!. You should see a page like below.

Voila! You are a genious. Now open the code editor and start hacking!

CLI Commands

Some useful commands to work with the project.

1. npm run dev to Start development server at http://localhost:3000

2. npm run build for Generating production build

3. npm serve To serve build files at http://localhost:9000

File Structure

Follow the steps below to setup your site template:

  1. Unzip the downloaded package and open the /azzle-next folder to find all the template files. You will need to upload these files to your hosting web server using FTP or localhost in order to use it on your website.
  2. Below is the folder structure and needs to be uploaded to your website or localhost root directory:
    • azzle-next/styles/css - Css files
    • azzle-next/app - All pages
    • azzle-next/public/images - Images files
    • azzle-next/app/page.js - Root file
  3. You should upload all or specific React Next Js files as per your need.
  4. You are good to go for adding your content now!
